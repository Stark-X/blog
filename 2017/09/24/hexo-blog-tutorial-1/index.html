Description

本系列教程目标是阅读者可以搭建起一个自己的博客在Github Page上。
进阶：利用Github的Webhook功能自动在VPS上发布自己的文章。

Hexo

什么是 Hexo？
Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

关于安装以及运行在本地的细节在Hexo官网已经有很详尽的中文文档描述，就不复制粘贴了。
Windows用户需要特别注意关于“windows用户”的提示。

Benefit

使用Hexo搭建博客足够简单，学习成本很低，只需要懂得使用Markdown，就可以写博客了，而且可以随时更换主题。
另外，在搭建、部署方面，因为Hexo的输出为纯静态网页，任何一个支持发布静态网页的服务都可以用于发布自己的博客，当然，如果有自己的VPS，也可以部署在上面，灵活性更高。以下列出我所知道的几个可以发布静态网页的平台：

  • GitHub Pages | 可通过*.github.io访问你的博客
  • 码云Pages | 可通过*.oschina.io访问你的博客

Prior Knoweledge

Must:

  • Markdown

Optional:

  • git
  • npm
  • Python / Javascript / Shell
  • Linux
